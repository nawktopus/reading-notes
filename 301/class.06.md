# Node.JS

## Intro to Node

1. What is node.js?

Node.js is a program that can execute JavaScript on our computers.

2. In your own words, what is Chrome’s V8 JavaScript Engine?

Chrome's V8 JavaScript Engine is an open-source JavaScript Engine that runs in chrome, allowing JavaScript to be able to run natively on our own computers.

3. What does it mean that node is a JavaScript runtime?

It means it is able to run JavaScript on multiple devices.

4. What is npm?

Npm stands for node package manager. It manages node and install anything node related.

5. What version of node are you running on your machine?

v18.11.0

6. What version of npm are you running on your machine?

8.19.2

7. What command would you type to install a library/package called ‘jshint’?

npm install -g jshint

8. What is node used for?

Node is to install and run tools to help automate the process of developing software.

## Pair Programming

1. What are the 6 reasons for pair programming?

Greater Efficiency, Engaged Collaboration, Learning from peers, Social skills, Job Interview Readiness, and work environment readiness.

2. In your experience, which of these reasons have you found most beneficial?

I've found peer and collaboration to be the best benefit from pair programming. Understanding from different perspectives to get the answer together.

3. How does pair programming work?

It works where one person is the driver and the other is navigator. One types and edits the code according to the navigator and vice versa.
